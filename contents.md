### Chapter 3 — Strings, Lists, Tuples, and Maps
*Розділ 3 — Рядки, списки, кортежи і словники*
> Основні структури даних Python

- Lists — Списки — впорядковані колекції елементів
- Tuples — Кортежи — незмінні списки

### Chapter 4 — Drawing with Turtles

- `if` / `elif` / `else` — Розгалуження: якщо… інакше якщо… інакше
- Comparison operators — `==`, `!=`, `>`, `<`, `>=`, `<=`
- Boolean values — `True` / `False`, `and` / `or` / `not`

### Chapter 6 — Going Loopy
*Розділ 6 — Цикли*
> Повторення дій без copy-paste

- `for` loop and `range()` — `for i in range(10):` — лічильний цикл
- `while` loop — Цикл «поки виконується умова»
- `break` and `continue` — Переривання та пропуск ітерацій

### Chapter 7 — Recycling Your Code with Functions and Modules

- Defining functions with `def` — Оголошення власної функції
- Arguments and return values — Вхідні параметри та повернення результату
- Modules and `import` — Файли з готовим кодом; `import`

---

### Chapter 8 — How to Use Classes and Objects
*Розділ 8 — Як використовувати класи та об'єкти*

- Classes and instances — Клас — «шаблон», об'єкт — «екземпляр»
- Attributes and methods — Атрибути (властивості) та методи (дії)
- Inheritance — Клас-нащадок отримує можливості батька

### Chapter 9 — Python's Built-In Functions

- Type conversions — `int()`, `str()`, `float()` — перетворення типів

### Chapter 10 — Useful Python Modules
*Розділ 10 — Корисні модулі Python*

- `sys` — `sys.exit()` та аргументи командного рядка

### Chapter 11 — More Turtle Graphics

### Chapter 12 — Using tkinter for Better Graphics

- Animation loop — `canvas.move()` + `after()` — анімація в циклі
