# Chapter 1 - Getting Started with Python

Welcome to your Python adventure.

## What is Python?

Python is a programming language used to make games, apps, websites, and tools.
It is popular because it is readable and beginner-friendly.

## What you learn in this chapter

- How to run your first Python code
- How `print()` works
- How to show text on one line and multiple lines
- How to use quotes correctly in strings

## Key idea

A program is a list of instructions.
Python reads your instructions and runs them step by step.

## Examples

### Printing text

```python
print("Hello, Python!")
```

This shows the text `Hello, Python!` on the screen.

### Printing multiple lines

You can print several lines at once using `\n` (the newline character):

```python
print("Line 1\nLine 2\nLine 3")
```

Or you can use several `print()` calls:

```python
print("Line 1")
print("Line 2")
```

### Using quotes

Strings are text inside quotes. You can use double quotes or single quotes:

```python
print("Hello")   # double quotes
print('Hello')   # single quotes — same result
```

If your text contains one type of quote, use the other type around it:

```python
print("My cat is named 'Pixel'.")
print('He said: "Python is cool!"')
```

## Quick tips

- Use `print()` to show something on the screen.
- Strings are text inside quotes: `"hello"` or `'hello'`.
- Use `\n` inside a string to start a new line.
- Mix single and double quotes to include quotes in your text.
- You can run code cell by cell in Jupyter or Colab.
