# Chapter 1 - Getting Started with Python

Welcome to your Python adventure.

## What is Python?

Python is a programming language used to make games, apps, websites, and tools.
It is popular because it is readable and beginner-friendly.

## What you learn in this chapter

- How to run your first Python code
- How `print()` works
- How to show text on one line and multiple lines
- How to use quotes correctly in strings

## Key idea

A program is a list of instructions.
Python reads your instructions and runs them step by step.

## Quick tips

- Use `print()` to show something on the screen.
- Strings are text inside quotes: `"hello"` or `'hello'`.
- You can run code cell by cell in Jupyter or Colab.
